App Setup Steps

  1. Login in to GoTo Delevoper Site Click here
  2. Get Authorization Code Click here
  3. Copy the Authorization Code which is redirected URL after code={AuthorizationCode}
  4. Paste the copied code in right side textarea and click on Validate AuthCode Button
  5. If Authorization Code is valid then you will be redirected to your Dashboard
